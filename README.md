# DePauw x HUB360 — Business Analytics Capstone

**Program**: DePauw University Business Analytics Capstone, Spring 2026
**Partner**: HUB360 LLC

## Overview

HUB360 is building a mobile platform that helps pharmaceutical companies manage compliant meal ordering for healthcare professional (HCP) interactions under the Sunshine Act. This capstone brings analytical perspectives to real product challenges across three workstreams: compliance visualization, analytics dashboards, and market research.

## Contributing

- **No real client data** — synthetic and public data only (CMS Open Payments is fine)
- **Keep it clean** — use clear file/folder names, include comments in code, and commit meaningful messages
- **Notebooks** — if using Jupyter, clear outputs before committing (keeps diffs readable)
- **Large files** — don't commit datasets over 50MB. Add them to `.gitignore` and document where to download them
